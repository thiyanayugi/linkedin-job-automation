# Data Directory

This directory should contain your resume PDF file.

## Setup

1. Add your resume PDF to this directory
2. Name it `resume.pdf` or update the `RESUME_PATH` in your `.env` file

Example:

```bash
cp /path/to/your/resume.pdf data/resume.pdf
```

## File Requirements

- **Format**: PDF only
- **Size**: Recommended under 5MB
- **Content**: Should contain your professional experience, skills, and qualifications

## Privacy Note

⚠️ Your resume file is excluded from Git by default (see `.gitignore`).
Never commit your actual resume to version control.
